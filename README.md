# Manufacturing Quality Performance Analytics

## Project Overview

This project analyzes manufacturing quality data to identify factors affecting product quality, production efficiency, manufacturing cost, and operational performance. The analysis was performed using SQL, Python, and Power BI to uncover quality trends and support data-driven decision making.

---

## Business Problem

Manufacturing companies need to reduce defects while maintaining production efficiency and controlling operational costs.

This project answers questions such as:

- Which plant has the highest production?
- Which shift produces more defects?
- Does machine age affect product quality?
- Which production line performs worst?
- Which inspection method detects the most defects?
- Which material grade requires additional quality checks?

## Key Insights

- Older machines showed significantly higher defect rates.
- Night shift recorded the highest defect rate.
- Grade C material showed higher quality issues than Grade A.
- Line C recorded the highest defect percentage.
- Longer rework time was associated with lower pass rates.
- Environmental conditions showed only a minor impact on defects compared to machine age.
